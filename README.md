# IBM Bob Projects - Homepage

A modern, minimal web application for managing project links.

## 🌟 Features

- ✨ Modern gradient UI design
- 🔗 Add and manage project links
- 💾 Automatic data persistence (localStorage)
- 📱 Fully responsive design
- 🚀 Zero dependencies - pure HTML/CSS/JS

## 🚀 Live Demo

Visit: [https://pmacse004.github.io/BobHomePage](https://pmacse004.github.io/BobHomePage)

## 📦 Deployment

### GitHub Pages
1. Push to GitHub
2. Go to Settings → Pages
3. Select `main` branch
4. Your site will be live at `https://pmacse004.github.io/BobHomePage`

### GoDaddy Hosting
1. Upload `index.html` to your hosting root directory
2. Access via your domain

## 💻 Local Development

Simply open `index.html` in your browser - no build process needed!

## 📝 Usage

1. Enter a link name and URL
2. Click "Add Link" or press Enter
3. Click any link to open it in a new tab
4. Use the Delete button to remove links
5. Your links are automatically saved

## 🎨 Customization

All styles are in the `<style>` section of `index.html`. Modify colors, fonts, or layout as needed.
